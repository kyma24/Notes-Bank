```jsx
do {
	code block
}
while (condition);
```

```jsx
var i = 20;
do {
	document.write(i + "<br />");
	i++;
}
while (i <= 25);
//just like in java, loop will always be executed at least once, even if condition is false, bc code block is executed before condition is tested.
```
