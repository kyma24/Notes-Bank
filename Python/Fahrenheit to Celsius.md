```py
# C = 5/9 (F - 32)
# F = (C \* 9/5) + 32
# Conversion from Fahrenheit to Celcius

#Get the Fahrenheit value
F = eval(input("Please enter a value for Fahrenheit."))

#Calculate the Celsius value
C = 5/9 \* (F - 32)

#Print out the result
print("The celsius value of ", F, "degrees Fahrenheit is ", C, ".")
```
